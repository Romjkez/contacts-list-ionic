# Книга контактов (Android & IOS & WEB)
Мобильная интеграция | Лабораторная №4

## Основной функционал

- Просмотр контактов
- Удаление контактов
- Редактирование контактов
- Добавление в избранное
- Удаление из избранного

Реализовано с помощью Ionic Framework
